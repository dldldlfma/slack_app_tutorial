# slack_app_tutorial

## 미리 설치해야 되는 것들
- npm
- ngrok
- flask
- slacker

## 사용방법

- 복잡하니 나중에 자세히 적자
- 우선은 api.slack.com 인가 거기서 app 만들고 권한주고 
- 통신할 수 있게 flask 서버 열고 거기에다가 @app.route("경로") 써서 명령 받을 수 있게 하고 
- ngrok http 8080 해서 slack에서 내 컴퓨터로 접속할 수 있게 주소를 뚫는다.
- 대충 뭐 그런방식으로 설계하고 하면된다.
